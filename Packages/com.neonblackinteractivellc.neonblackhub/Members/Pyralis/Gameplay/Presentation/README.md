# Pyralis Presentation

This domain owns cross-feature presentation systems such as animation routing, camera infrastructure, visual feedback helpers, and shared visual adapters.

Current major surfaces:

- `Animation/` - `ActorAnimationDriver` and animation signal routing.
- `Camera/` - `CinemachineCameraRigController`, 2D camera aspect helpers, camera shake, and occlusion helpers.
- `Visuals/` - actor shadows, billboarding, sprite/text flashers, and shared camera shake support.
